# Student Performance Data Analysis

## Project Overview
This project analyzes student performance using Python.

## Technologies Used
- Python
- Pandas
- Matplotlib

## Dataset
student_data.csv

## Features
- Data Loading
- Data Cleaning
- Missing Value Check
- Duplicate Check
- Average Marks
- Highest Marks
- Topper Analysis
- City-wise Analysis
- Gender-wise Analysis
- Graphs
- Correlation Analysis

## Graphs
1. City Bar Chart
2. City Pie Chart
3. Math Line Graph
4. Top 5 Students Graph
5. Attendance Graph
6. Correlation Heatmap

## Conclusion
The project analyzes students' academic performance and attendance using Python data analysis libraries.
